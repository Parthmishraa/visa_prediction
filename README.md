## 📝 Description

AI-Enabled Visa Status Prediction is a sophisticated machine learning solution designed to demystify the visa application process and provide clarity to applicants worldwide.  

Built using Python, this system analyzes historical application data to deliver accurate predictions regarding processing times and status outcomes.  

By transforming complex data into actionable insights, the project aims to improve transparency, reduce applicant uncertainty, and facilitate more informed decision-making for individuals and organizations navigating the complexities of global travel and immigration.

## 📊 Model Overview

The project follows a complete machine learning pipeline to predict visa status:

-  **Data Preprocessing:** Handling missing values, encoding categorical variables, and data cleaning  
-  **Exploratory Data Analysis (EDA):** Understanding patterns, trends, and relationships in data  
-  **Feature Engineering:** Selecting and transforming relevant features for better model performance  
-  **Model Training:** Applying machine learning algorithms for prediction  
-  **Model Evaluation:** Evaluating performance using metrics like Accuracy, Precision, Recall, and F1-score  
-  **Deployment:** Integrating the trained model into a Streamlit web application for real-time predictions
